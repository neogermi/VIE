VIE widgets

There are several user interface widgets built on top of VIE. You can easily drop these widgets into your web application to gain immediate benefit of semantic interaction:

  • Create — inline content editing tool
  • VIE autocomplete.js — easy autocomplete for VIE entities
  • Image Search — semantic search for images
  • Auto Tagger — automatic tagging of content
  • Annotate.js — semi-automatic content annotation tool
  • Infobox widget — Configurable widget visualisation widget with a configurator

Integrating VIE

In nutshell, integrating VIE into your web application is as easy as:

  1. Mark up your pages with RDFa annotations
  2. Include VIE into the pages
  3. Implement Backbone.sync

Learn more on the VIE mailing list.
